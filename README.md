# Dataknac

A premium, editorial-grade digital experience for a strategic intelligence firm. Designed with dark themes, rigorous grid systems, and motion physics.

## Project Structure

- **Core**: React 19, vite
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion, Spline, Spotlight
- **Typography**: Inter (Sans), Playfair Display (Serif)

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Features

- **3D Hero Section**: Interactive Spline scene with spotlight effects.
- **Hash Navigation**: Custom smooth scrolling router implementation.
- **Dynamic Intelligence**: Real-time data visualization components.
- **Responsive Design**: Mobile-first approach with complex grid layouts.
